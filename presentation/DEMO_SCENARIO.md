# RE:OUT 데모 시연 시나리오

## 🎬 준비물

### 의류 사진 3장
1. **깨끗한 옷** (A등급 예상)
   - 예: 깔끔한 코트, 새 셔츠
   - 목적: 높은 재판매 가치 보여주기

2. **보통 옷** (B등급 예상)
   - 예: 조금 사용감 있는 티셔츠
   - 목적: 기부 추천 보여주기

3. **낡은 옷** (C등급 예상)
   - 예: 변색된 청바지
   - 목적: 업사이클/재활용 추천

## 🎯 시나리오 1: 메인 시연 (90초)

### 사진: ZARA 코트 (A등급)

#### 1단계: 업로드 (10초)
```
동작: 메인 화면 → "사진 업로드하기" 클릭 → 파일 선택

멘트:
"이것이 RE:OUT의 메인 화면입니다.
사진 업로드 버튼 하나만 있습니다.
정말 간단하죠?"
```

#### 2단계: 분석 중 (5초)
```
동작: 로딩 화면 표시

멘트:
"지금 Google Cloud Vision AI가
실시간으로 이 옷을 분석하고 있습니다.
약 3-5초 소요됩니다."

(로딩 메시지 읽기)
"AI가 카테고리를 파악하는 중..."
```

#### 3단계: 결과 확인 (40초)
```
동작: 결과 화면 스크롤하며 설명

멘트:
"분석이 완료되었습니다!

[이미지 & 기본 정보]
AI가 이 의류를 '아우터 - 코트'로 분류했고,
상태는 A등급, '사용감 거의 없음'으로 평가했습니다.

[브랜드 인식]
ZARA 브랜드도 자동으로 인식했습니다.
이건 Google Vision의 OCR 기능입니다.

[환경 임팩트]
만약 이 옷을 재사용하면,
CO2 3kg 절감, 나무 0.14그루 심는 효과,
물 2,700리터를 절약할 수 있습니다.

[5가지 경로]
자, 이제 가장 중요한 부분입니다.
RE:OUT은 5가지 경로를 우선순위별로 추천합니다.

1순위는 '재판매'입니다.
예상 판매가는 45,000~65,000원이고,
당근마켓, 번개장터 같은 플랫폼을 추천합니다.

2순위는 렌탈 서비스, 3순위는 기부...
이렇게 모든 옵션을 한 눈에 볼 수 있습니다."
```

#### 4단계: 빠른 재시연 (15초)
```
동작: 다시 분석하기 → 다른 사진 업로드

멘트:
"다른 사진도 빠르게 보여드리겠습니다.
(업로드) → (분석) → (결과)
이번엔 B등급으로 나왔고, 기부가 1순위네요."
```
