### 2. [user-scenarios.md](user-scenarios.md) - 사용자 시나리오
**실제 사용 상황**을 구체적으로 묘사한 7가지 시나리오입니다.

**시나리오 목록:**

1. **개인 사용자 - 옷장 정리하는 직장인 지민**
   - 이사 준비 중 27벌의 옷 처리
   - 30분 만에 분석 완료, 32만원 수익

2. **B2B 기업 - 중소 패션 브랜드 '어반베이직'**
   - 시즌 오프 재고 600벌 처리
   - 2,525만원 비용 개선 효과

3. **업사이클링 스튜디오 - '새옷'**
   - 원료 소싱 시간 66% 감소
   - 월 80-120벌 안정적 공급 확보

4. **중고 거래 플랫폼 - 'K중고'의 API 연동**
   - 가격 추천 기능으로 판매 성공률 45% 향상
   - 월 500만 사용자에게 브랜드 노출

5. **환경 단체 - '그린패션'의 캠페인 활용**
   - 3만 명 참여, 15만 벌 분석
   - 300톤 CO2 절감 성과

6. **명품 재판매 - 대학생 민지의 버버리 코트**
   - 10년 된 명품 코트를 40만원에 판매
   - 재판매 가치 학습 및 소비 패턴 변화

7. **대기업 - S패션의 ESG 프로그램**
   - 12만 고객 참여, 35만 벌 회수
   - 1,050톤 CO2 절감, ESG 투자 300억 유치
