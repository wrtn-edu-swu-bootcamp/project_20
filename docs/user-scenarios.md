# RE:OUT 사용자 시나리오

실제 사용 상황을 구체적으로 묘사한 시나리오입니다. 각 페르소나가 RE:OUT을 어떻게 사용하고, 어떤 가치를 얻는지 보여줍니다.

---

## 시나리오 1: 개인 사용자 - 옷장 정리하는 직장인 지민

### 배경
- **이름**: 김지민 (29세, 여성)
- **직업**: IT 기업 마케터
- **상황**: 이사를 앞두고 5년간 모은 옷 정리 중
- **고민**: "입지 않는 옷이 너무 많은데, 뭘 버리고 뭘 팔아야 할지 모르겠어. 하나하나 중고나라에 올리기엔 시간도 없고..."

### RE:OUT 사용 과정

#### 1단계: 앱 발견 및 설치
- 토요일 오전, 옷장 앞에서 막막함을 느낌
- "옷 처리" 검색 → RE:OUT 발견
- "사진으로 3초 분석" 문구에 호기심 → 앱 설치

#### 2단계: 첫 의류 분석
- **대상**: 3년 전 구매한 자라(ZARA) 코트, 3번 정도만 착용
- 앱 실행 → "옷 분석하기" 버튼 클릭
- 코트를 바닥에 펼쳐놓고 사진 촬영
- 3초 후 결과 화면:
  - 카테고리: 아우터 - 코트
  - 상태 등급: A등급 (사용감 거의 없음)
  - 손상: 없음
  
**추천 경로:**
1. **재판매** - 예상가: 45,000~65,000원 (새 제품 가격의 30-40%)
   - 당근마켓, 번개장터 추천
   - "비슷한 제품이 최근 55,000원에 거래되었어요"
2. **기부** - 아름다운가게 (집에서 500m)
3. **업사이클링** - 리메이크 업체 연결 가능
4. **렌탈 서비스** - 시즌 의류 렌탈 업체 매입 가능
5. **재활용** - 비추천 (상태 양호)

#### 3단계: 결정 및 실행
- "어? 생각보다 비싸게 팔 수 있네!"
- "재판매" 선택 → 당근마켓 앱 자동 실행
- RE:OUT 추천가를 참고하여 58,000원에 등록
- 3일 후 판매 완료

#### 4단계: 나머지 의류 처리
- 총 27벌 분석 (30분 소요)
- 결과:
  - **재판매** (10벌): 예상 수익 약 35만원
  - **기부** (12벌): 아름다운가게에 일괄 기부
  - **재활용** (5벌): 동네 의류수거함에 배출

### 결과 및 만족도
- **시간 절약**: 각 옷마다 3초 분석, 고민 시간 제로
- **경제적 이익**: 실제 판매 수익 32만원 (예상의 91%)
- **환경 기여**: CO2 약 75kg 절감 (앱 통계에 표시)
- **만족도**: "완전 신세계! 친구들한테 다 추천했어요. 옷 정리가 이렇게 쉬울 줄이야."

### 후속 사용
- 이후 새 옷 구매 전 "나중에 RE:OUT으로 팔 수 있을까?" 고려
- 분기마다 옷장 정리 시 RE:OUT 사용
- SNS에 환경 기여 인증 공유 → 친구 3명 유입

## 시나리오 2: B2B 기업 - 중소 패션 브랜드 '어반베이직'

### 배경
- **기업**: 어반베이직 (직원 15명, 연 매출 30억)
- **담당자**: 박현수 대리 (경영지원팀)
- **상황**: 2024 F/W 시즌 종료, 미판매 재고 약 600벌
- **고민**: 
  - 재고 보관 비용 월 200만원
  - 폐기 비용 벌당 3,000원 = 180만원
  - "다른 방법이 없을까? ESG 보고서에 뭘 써야 하지?"

### RE:OUT 도입 과정

#### 1단계: 서비스 발견
- 환경부 '순환경제 지원 사업' 공고에서 RE:OUT 발견
- 웹사이트 방문 → "B2B 무료 파일럿" 신청
- 영업팀과 미팅: "600벌 무료 분석 제공, 효과 검증 후 유료 전환"

#### 2단계: 데이터 준비
- 재고 목록 Excel 작성 (제품코드, SKU, 사이즈 등)
- 각 제품 사진 촬영 (인턴 2명, 2일 소요)
- 파일 정리: Excel + 이미지 폴더 (파일명으로 매칭)

#### 3단계: 일괄 분석
- RE:OUT 대시보드에 로그인
- "일괄 업로드" → CSV + 이미지 폴더 업로드
- 검증 완료 → "분석 시작" 버튼
- 600벌 분석 완료 (약 15분)

#### 4단계: 결과 확인
**전체 요약 리포트:**
- S등급 (신품): 180벌 (30%)
- A등급: 250벌 (42%)
- B등급: 120벌 (20%)
- C등급: 50벌 (8%)

**추천 경로 분포:**
1. **재판매** (430벌, 72%)
   - 예상 수익: 약 1,800만원 (도매가의 40-60%)
   - 추천: 중고 플랫폼, 아울렛 위탁 판매
   
2. **기부** (120벌, 20%)
   - 세액 공제 예상: 약 150만원
   - 추천: 굿윌스토어 (B등급 이상 수령 가능)
   
3. **업사이클링** (30벌, 5%)
   - 업사이클 브랜드 '래;코드' 원료 매입 제안
   - 매입가: 벌당 5,000원 = 15만원
   
4. **재활용** (20벌, 3%)
   - 섬유 재활용 업체 연결
   - 처리 비용: 무료 (업체 직접 수거)

#### 5단계: 실행 계획 수립
**파트너사 선택:**
- 재판매: 무신사 스토어 중고관, 번개장터 프로 입점
- 기부: 굿윌스토어와 MOU 체결
- 업사이클링: 래;코드와 정기 공급 계약

**예상 효과:**
- 수익/절감: 1,800만원 (재판매) + 150만원 (세액공제) + 15만원 (업사이클) + 180만원 (폐기 비용 절감) = **2,145만원**
- 기존 손실 대비: 380만원 지출 → 2,145만원 수익 = **2,525만원 개선**
- CO2 절감: 약 1.8톤 (RE:OUT 리포트 제공)

#### 6단계: 처리 완료
- 2주 내 재판매 상품 등록 완료
- 1개월 내 85% 판매 완료 (실제 수익 1,520만원)
- 기부/업사이클 처리 완료

### 결과 및 효과

**정량적 효과:**
- **실제 수익**: 약 1,900만원 (예상의 89%)
- **시간 절약**: 수작업 대비 80% 절감 (예상 2주 → 실제 3일)
- **비용 절감**: 보관비 + 폐기비 절감

**정성적 효과:**
- **ESG 보고서**: 구체적 수치와 리포트 활용
- **브랜드 이미지**: SNS 홍보 ("버리지 않는 브랜드")
- **내부 만족도**: 직원들의 환경 의식 향상

**후속 계약:**
- RE:OUT Pro 플랜 연간 계약 (월 20만원)
- 분기마다 재고 분석 정례화
- 다른 브랜드에 RE:OUT 추천

## 시나리오 3: 업사이클링 스튜디오 '새옷'

### 배경
- **기업**: 업사이클링 브랜드 '새옷' (직원 5명)
- **대표**: 이수진 (디자이너 출신)
- **상황**: 양질의 헌 옷 원료 확보에 어려움
- **고민**: "좋은 원단을 찾으려면 중고 시장을 하루 종일 돌아다녀야 해. 비효율적이고 원가가 너무 높아."

### RE:OUT 활용 방식

#### 기존 방식의 문제
- **원료 소싱**: 중고 매장, 기부처 방문 → 수작업 선별
- **시간**: 주 2-3일 소요
- **원가**: 원료비 + 인건비 + 교통비 = 제품 원가의 30%
- **품질**: 좋은 원단/디자인 발견 불확실

#### RE:OUT 도입 후

**1. 공급자 네트워크 연결**
- RE:OUT 파트너 포털 등록
- "업사이클 원료 매입" 업체로 등록
- 관심 카테고리: 데님, 가죽, 빈티지 디자인

**2. 자동 매칭**
- RE:OUT 사용자가 의류 분석 시
- "업사이클링" 경로 추천되면 '새옷' 자동 매칭
- 사용자가 "연락 받기" 동의 시 정보 전달

**3. 원료 확보**
- 주 평균 20-30벌 제안 받음
- 사진과 분석 결과로 사전 검토
- 원하는 원료만 선택 → 택배 수거 또는 직접 방문
- 매입가: 벌당 3,000~10,000원 (상태/소재별)

**4. 역으로 RE:OUT 활용**
- 받은 원료 중 사용하지 않는 것
- RE:OUT으로 재분석 → 다른 경로 추천
- "원료의 원료"도 낭비하지 않음

### 결과 및 효과

**정량적 효과:**
- **소싱 시간**: 주 3일 → 1일 (66% 감소)
- **원가 절감**: 제품 원가 30% → 20% (10%p 개선)
- **공급 안정성**: 월 평균 80-120벌 안정적 확보
- **품질 향상**: 사전 필터링으로 적합도 90%

**정성적 효과:**
- **디자인 집중**: 소싱 시간 감소 → 디자인에 더 많은 시간 투자
- **스토리텔링**: "RE:OUT 파트너"로 마케팅, 브랜드 신뢰도 향상
- **커뮤니티**: RE:OUT 통해 의류 제공한 고객과 연결, 완제품 할인 제공

**수진 대표 인터뷰:**
"RE:OUT은 저희에게 안정적인 공급망이에요. 이제 좋은 원단을 찾아 헤매지 않아도 되고, 고객들도 자신의 옷이 새로운 제품으로 다시 태어나는 걸 보며 뿌듯해하세요. 우리 모두에게 윈윈이죠."

## 시나리오 4: 중고 거래 플랫폼 'K중고'의 API 연동

### 배경
- **플랫폼**: K중고 (월간 사용자 500만명)
- **담당자**: 개발팀 김태영 팀장
- **상황**: 사용자들이 가격 책정에 어려움 호소
- **목표**: RE:OUT API 연동으로 사용자 경험 개선

### 연동 과정

#### 1단계: API 계약
- RE:OUT API 서비스 설명 듣기
- 가격: 월 1만 건 50만원, 10만 건 400만원
- 테스트 계정 발급 → 2주간 테스트

#### 2단계: 기능 통합
**K중고 앱 업데이트:**
- 중고 물품 등록 화면에 "AI 가격 추천" 버튼 추가
- 사용자가 버튼 클릭 시:
  1. 업로드한 사진을 RE:OUT API로 전송
  2. 3초 후 분석 결과 수신
  3. 상태 등급, 예상 가격대 자동 입력
  4. 사용자가 수정 가능

#### 3단계: 사용자 반응
- 런칭 첫 주: 5만 건 사용
- 평균 등록 시간: 8분 → 3분 (62% 단축)
- 가격 적정성: AI 추천가 사용 시 판매 성공률 45% 향상
- 앱 리뷰: "이제 가격 고민 안 해도 돼요!" (평점 4.2 → 4.6)

#### 4단계: 추가 기능
**RE:OUT 데이터 활용:**
- K중고 내부 데이터 + RE:OUT 분석 결합
- 더 정확한 시세 예측 모델 개발
- "이 가격이면 3일 내 판매 확률 85%입니다" 메시지 제공

### 결과 및 효과

**K중고 입장:**
- **사용자 만족도**: 크게 향상 (NPS +15점)
- **거래 성사율**: 12% 증가
- **체류 시간**: 등록 과정 단축으로 이탈률 감소
- **차별화**: 경쟁 플랫폼 대비 독보적 기능

**RE:OUT 입장:**
- **API 수익**: 월 400만원 (10만 건)
- **브랜드 노출**: 월 500만 사용자에게 "Powered by RE:OUT" 노출
- **데이터 확보**: K중고 거래 데이터로 모델 정확도 향상

## 시나리오 5: 환경 단체 '그린패션'의 캠페인 활용

### 배경
- **단체**: 환경 NGO '그린패션'
- **캠페인**: "옷 한 벌 살리기" 프로젝트
- **목표**: 의류 폐기 줄이기 대중 캠페인

### RE:OUT 협업

#### 캠페인 기획
"옷장 속 잠자는 옷 찾기 챌린지"
- 기간: 1개월
- 참여 방법:
  1. RE:OUT 앱으로 안 입는 옷 분석
  2. 추천 경로대로 처리 (증명 사진)
  3. SNS 인증 (#옷한벌살리기)
- 보상: 추첨으로 친환경 제품 증정

#### RE:OUT 지원
- **무료 크레딧**: 캠페인 참여자 무제한 분석 무료
- **특별 페이지**: 캠페인 전용 인터페이스
- **통계 제공**: 실시간 환경 기여도 집계 대시보드

#### 결과
- **참여자**: 3만 명
- **분석 의류**: 약 15만 벌
- **실제 처리**: 약 10만 벌 (66%)
- **CO2 절감**: 약 300톤 (나무 15,000그루 효과)
- **미디어 노출**: 주요 언론 20여 곳 보도

### 효과

**그린패션 입장:**
- 구체적 성과 수치로 캠페인 효과 입증
- RE:OUT 데이터로 정책 제안서 작성
- 정부 지원금 확보

**RE:OUT 입장:**
- 대규모 사용자 유입 (3만 명)
- 긍정적 브랜드 이미지 구축
- 환경 임팩트 가시화

**사회적 효과:**
- 의류 재사용 문화 확산
- 개인의 작은 실천이 큰 변화를 만든다는 인식

## 시나리오 6: 명품 코트를 물려받은 대학생 민지

### 배경
- **이름**: 이민지 (23세, 여성)
- **상황**: 어머니에게 10년 된 버버리 코트를 물려받음
- **고민**: "엄마 옷이라 스타일이 안 맞는데... 그냥 버리기엔 아깝고, 팔면 얼마나 받을 수 있을까?"

### RE:OUT 사용

#### 분석 결과
- **카테고리**: 아우터 - 트렌치 코트
- **브랜드**: 버버리 (Burberry) 인식
- **상태 등급**: B등급 (사용감 있으나 양호)
- **손상**: 경미한 보풀, 일부 변색

**추천 경로:**
1. **재판매 (명품 전문)** - 예상가: 350,000~500,000원
   - 머스트잇 리셀, 발란 추천
   - "버버리 빈티지 트렌치는 인기 아이템입니다"
2. **일반 재판매** - 예상가: 250,000~350,000원
3. **기부** - 명품 기부 프로그램

## 시나리오 7: 대기업 패션 브랜드의 ESG 프로그램

### 배경
- **기업**: 국내 대형 패션 그룹 'S패션' (연 매출 3조원)
- **담당 부서**: ESG 경영팀
- **압박**: 2025년 ESG 공시 의무화, 구체적 실적 필요
- **목표**: 의류 회수 및 재활용 프로그램 구축

### RE:OUT 전사 도입

#### 1단계: White Label 계약
- RE:OUT 기술을 자사 브랜드로 커스터마이징
- 'S패션 순환' 앱/웹 런칭
- 뒷단은 RE:OUT 엔진 사용 (API 연동)

#### 2단계: 고객 대상 프로그램
**"S패션 옷 돌려주기"**
- 자사 브랜드 의류를 RE:OUT으로 분석
- 상태에 따라 포인트 차등 지급
  - S등급: 50,000P (5만원 상당)
  - A등급: 30,000P
  - B등급: 10,000P
  - C등급: 5,000P
- 포인트로 신제품 구매 가능

#### 3단계: 회수 의류 처리
- RE:OUT 분석 결과 기반 자동 분류
- 처리 방식:
  - S~A등급: 자사 중고 플랫폼 '리:S' 재판매
  - B등급: 제휴 기부처 기증
  - C~D등급: 자사 업사이클 라인 'S:Cycle' 원료

#### 4단계: 연간 성과 (1년 후)
- **참여 고객**: 12만 명
- **회수 의류**: 35만 벌
- **재판매**: 15만 벌 (45억원 매출)
- **기부**: 12만 벌
- **업사이클**: 8만 벌 (새 제품 라인 출시)
- **CO2 절감**: 1,050톤

### ESG 보고서 효과
- **구체적 수치**: RE:OUT 제공 데이터 전면 활용
- **외부 인증**: 탄소 감축 제3자 검증
- **투자 유치**: ESG 펀드 300억원 유치
- **브랜드 가치**: 환경 책임 기업 이미지, 주가 상승

### S패션 ESG팀장 인터뷰
"RE:OUT 덕분에 ESG를 '해야 하는 일'에서 '가치를 만드는 일'로 바꿨습니다. 고객들도 좋아하고, 실제 환경에도 기여하고, 사업적으로도 의미가 있어요. 이보다 좋은 파트너십은 없습니다."

RE:OUT의 강점은 **개인부터 대기업까지 모두에게 가치**를 제공한다는 점입니다.

|------------|----------|-----------------|--------|
